# 扩展安装说明

## 本地加载

1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择项目中的 [extension](.) 目录

## 配置步骤

1. 点击扩展图标，打开弹窗
2. 填写服务器地址，例如 `http://localhost:3000`
3. 点击“测试连接”
4. 保存设置
5. 如需自动同步，打开相关同步开关
6. 首次使用建议手动执行一次全量同步

## 建议验证

1. 在浏览器中新建一个书签，确认网站能同步出现
2. 删除该书签，确认网站也同步删除
3. 移动一组书签到新文件夹，确认网站位置同步更新
4. 在普通网页按 `Ctrl+Space`，确认搜索浮层可正常打开
5. 打开网站控制中心，点击“合并书签”，确认重复项可被手动合并

## 打包发布

如果需要生成发布包，请在项目根目录运行：

```bash
npm run build-extension
```

生成结果：

- `extension.zip`

## 注意事项

- 如果是远程服务器，请确认服务端已重启到最新版本，以便 `/api/favicon`、实时同步与全量对账逻辑生效
- 如果两台设备需要实时同步，网站页面最好都保持在线打开
- 更新扩展代码后，请在扩展管理页重新加载扩展
