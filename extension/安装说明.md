# God's Bookmark 扩展安装说明

## 安装前准备

请先确认：

- God's Bookmark 网站服务已经可访问
- 你知道服务器地址，例如 `http://localhost:3000` 或部署域名
- 浏览器为 Chrome 或 Edge

## Chrome 安装

1. 打开 `chrome://extensions/`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择项目中的 `extension` 目录
5. 确认扩展图标已经出现在工具栏

## Edge 安装

1. 打开 `edge://extensions/`
2. 开启“开发人员模式”
3. 点击“加载解压缩的扩展”
4. 选择项目中的 `extension` 目录

## 首次配置

1. 点击扩展图标打开弹窗
2. 输入服务器地址
3. 点击“测试连接”
4. 保存设置
5. 如需自动同步，打开相关同步开关
6. 首次使用建议手动执行一次全量同步

## 推荐验证步骤

1. 在浏览器书签栏中新建一个测试书签
2. 刷新 God's Bookmark 页面
3. 确认网站端出现对应书签
4. 在网站中删除这个测试书签
5. 确认本机浏览器书签也同步删除
6. 在普通网页按 `Ctrl+Space`，确认搜索浮层可正常打开

## 打包发布

如果需要生成发布包，请在项目根目录运行：

```bash
npm run build-extension
```

输出文件为：

- `extension.zip`

## 更新扩展

当代码更新后：

1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 找到 God's Bookmark 扩展
3. 点击“重新加载”
4. 刷新已打开的网站页面

## 常见提醒

- 如果浏览器提示“扩展上下文已失效”，通常是扩展刚被重新加载，刷新网页即可。
- 如果是远程服务器，请确认服务器已重启到最新版本，以便 `/api/favicon` 和实时同步接口生效。
- 如果两台设备需要实时同步，网站页面最好都保持在线打开状态。
