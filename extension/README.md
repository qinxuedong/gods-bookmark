# God's Bookmark 书签同步扩展

自动将浏览器书签同步到 God's Bookmark 书签模块的 Chrome/Edge 扩展。

## 功能特点

- ✅ 自动同步：在浏览器中添加书签时，自动同步到网站
- ✅ 文件夹分类：根据浏览器书签栏的文件夹自动分类
- ✅ 实时同步：支持创建、更新、删除三种同步模式
- ✅ 简单配置：通过扩展弹出窗口轻松管理设置

## 安装方法

### 方法一：开发者模式安装（推荐用于开发）

1. 打开 Chrome/Edge 浏览器
2. 访问扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `extension` 文件夹
6. 扩展安装完成！

### 方法二：打包安装（用于分发）

1. 在扩展管理页面点击"打包扩展程序"
2. 选择 `extension` 文件夹
3. 生成 `.crx` 文件
4. 通过"加载已解压的扩展程序"安装

## 使用方法

### 首次使用

1. 安装扩展后，点击浏览器工具栏的扩展图标
2. 在弹出窗口中配置服务器地址（默认：http://localhost:3000）
3. 点击"保存设置"
4. 点击"测试连接"确保服务器正常运行
5. 勾选"启用自动同步"

### 自动同步

启用后，当您：
- 在浏览器中添加新书签 → 自动添加到网站的对应分类
- 修改书签名称或URL → 自动更新网站书签（需启用"书签更新时同步"）
- 删除浏览器书签 → 自动从网站删除（需启用"书签删除时同步"）

### 分类规则

- **文件夹书签**：根据浏览器书签栏中的文件夹名称创建分类
- **根目录书签**：保存到"浏览器书签"分类（可在设置中修改）

## 配置说明

### 基本设置

- **服务器地址**：God's Bookmark 服务器的地址（默认：http://localhost:3000）
- **默认分类名称**：根目录书签保存的分类名称（默认：浏览器书签）

### 同步选项

- **新书签时自动同步**：添加新书签时立即同步
- **书签更新时同步**：修改书签时同步更新
- **书签删除时同步**：删除书签时同步删除

## 注意事项

1. **服务器运行**：确保 God's Bookmark 服务器正在运行
2. **网络访问**：扩展需要访问本地服务器，请确保防火墙允许
3. **重复书签**：相同URL的书签会被更新而不是重复添加
4. **文件夹处理**：扩展会跳过文件夹，只同步实际的书签链接

## 故障排除

### 连接失败

- 检查服务器是否运行（`node server.js`）
- 检查服务器地址是否正确
- 检查防火墙设置

### 同步不工作

- 确认"启用自动同步"已勾选
- 检查对应的同步选项是否启用
- 查看扩展后台页面的控制台日志（`chrome://extensions/` → 查看视图 → 背景页）

### 书签未分类

- 确认书签添加到了书签栏的文件夹中
- 根目录的书签会保存到"浏览器书签"分类

```
├── background.js       # 后台服务脚本（监听书签变化）
├── popup.html          # 弹出窗口HTML
├── popup.js            # 弹出窗口脚本
├── options.html        # 选项页面HTML
├── options.js          # 选项页面脚本
└── README.md           # 说明文档
```

### API 端点

扩展使用以下 API 端点与服务器通信：

- `POST /api/bookmark/sync` - 同步书签到服务器
  - 请求体：`{ category, bookmark: { name, url, icon }, action: 'created'|'updated'|'removed' }`

### 权限说明

- `bookmarks`：访问浏览器书签
- `storage`：保存扩展配置
- `host_permissions`：访问本地服务器
