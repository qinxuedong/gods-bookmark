# Release Notes v2.0.0

发布日期：`2026-04-08`

## 相对 v1.1.2 的正式版变化

- 全局设置中的书签卡片自由布局能力完成一轮稳定化，修复了点击高亮、左右联动、拖动偏移、拖动变慢、边界越界和关闭设置后的残留缩放条
- 卡片尺寸与位置在编辑过程中更稳定，关闭全局设置后不再依赖刷新才能恢复正常显示
- 书签网格背景从“仅设置时显示”调整为常驻显示，网格透明度下调到 `4%`，外层虚线边框保持 `18%`，并修正了顶部边界与网格对齐关系
- 扩展与页面之间的同步链路增强了上下文失效保护，减少 `Extension context invalidated` 相关报错对页面交互的影响
- 导入后的首屏展示与 favicon 策略进一步优化，降低大批量导入时的等待感
- 页面 favicon 明确绑定为扩展内 `16x16` 图标，控制中心按钮图标资源完成更新

## 本次发布重点

- Web 应用与浏览器扩展版本统一升级到 `2.0.0`
- 本版本定位为相对 `1.1.2` 的正式大版本整理与发布
- 重点覆盖布局编辑、扩展同步健壮性与视觉交互一致性

## 受影响文件范围

- `js/app.js`
- `js/dashboard-layout.js`
- `extension/content.js`
- `css/style.css`
- `index.html`

## 发布说明

- Web 应用版本：`2.0.0`
- 浏览器扩展版本：`2.0.0`
- 扩展发布包可通过 `npm run build-extension` 重新生成
- 升级后建议重新加载浏览器扩展，并刷新已打开的书签首页标签页
