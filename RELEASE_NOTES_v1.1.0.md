# Release Notes v1.1.0

发布日期：`2026-04-04`

## 重点更新

- 修复 `Ctrl+Space` 在不同页面上行为不一致的问题，统一为直接打开搜索浮层
- 补齐浏览器书签新增、删除、移动、文件夹新增、文件夹删除的同步链路
- 支持双设备同时在线时的网站实时同步
- 全局设置中的书签模块支持拖拽调整尺寸与排序，并自动保存
- 扩展和网站统一切换到服务端 `/api/favicon` 图标接口，解决远程部署后图标缺失问题
- 新增控制中心“合并书签”手动工具，按“名称 + URL”规则合并重复项
- 合并书签后会同步回写浏览器，并在必要时自动执行一轮全量对账
- 更新扩展图标资源，补齐 `16 / 48 / 128 / 256` 多尺寸图标与 `icon.ico`

## 扩展版本

- `1.1.0`

## 发布说明

- 发布包通过 `npm run build-extension` 生成
- 发布前建议完成一次双机在线同步实测
- 更新后请重启服务端并重新加载扩展
