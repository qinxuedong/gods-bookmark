# Release Notes v1.1.2

发布日期：`2026-04-05`

## 本次更新

- 修复合并书签需要多次执行才能完成的问题
- 合并书签增加对比弹窗，支持按分类与来源文件夹手动选择删除项
- 修复合并后名称、网址与来源信息被错误覆盖的问题
- 修复书签卡片宽度偶发超出容器导致左右裁切的问题
- 优化卡片内拖动书签排序，减少整块重渲染和实时流回刷造成的跳动
- 修复导入 HTML 书签时未按原分类完整落库的问题
- 修复导入书签后未实时同步到浏览器后台的问题
- 修复谷歌书签管理器后台批量删除后，网页与服务器未实时同步的问题
- 优化浏览器后台删除场景，新增删除后的浏览器到服务器全量对账兜底

## 扩展版本

- `1.1.2`

## 发布说明

- Web 应用与浏览器扩展版本统一更新到 `1.1.2`
- 发布包通过 `npm run build-extension` 生成
- 更新后请重启服务端，并在 `chrome://extensions` 或 `edge://extensions` 中重新加载扩展
- 建议发布前重点验证：
  1. 合并书签弹窗与手动勾选删除
  2. 同卡片内拖动书签排序
  3. 导入带嵌套目录的 HTML 书签
  4. 谷歌书签管理器后台批量删除后的实时同步
